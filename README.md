#Projeto da disciplina de Projeto de Software  
Alunos: Lucas Tadeu e Esther de Freitas  
Professor: Baldoino Fonseca

PT-BR
Nosso módulo é focado nos anúncios da rede social do IC. Utilizaremos informações dos perfis dos usuários, campos onde o usuário marca se gostaria de ser voluntário e em quais setores, e se gostaria de receber recomendações de lojas e quais lojas lhe seriam interessantes. Com estes dados podemos encontrar ONGs e produtos que seriam atraentes ao perfil do usuário. Na prática utilizaremos as palavras chaves dessa parte do perfil do usuário e das classificações das ONGs / produtos, e selecionaremos algumas para serem apresentadas ao usuário, com uma frequência pré-definida por este.
